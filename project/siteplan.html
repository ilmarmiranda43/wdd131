Website Planning Document

WDD 131 • Week 05 • Project Site Plan

1) Site Name

SP Code Trails

I chose this name because the site guides beginners through practical “trails” of web learning (HTML, CSS, and JavaScript) with short, hands-on challenges and a friendly tone.

Optional domain idea: spcodetrails.org

2) Site Purpose

This site helps beginners practice web fundamentals with curated mini-guides and weekly challenges. Visitors can explore learning guides, view upcoming practice events, and save favorite challenges.

3) Scenarios

  • Which beginner JavaScript challenges can I do in 15 minutes and save as favorites?
  • How do I find the next practice session and sign up to receive reminders?

4) Color Schema

The site uses two main colors + neutrals.

Primary: #1F6FEB

Used for headings, buttons, links, and accents.

Secondary: #0E1116

Used for header/footer background and high-contrast text blocks.

Note: This color schema is applied on this siteplan document, too.

5) Typography

  • Headings: system-ui, -apple-system, Segoe UI, Roboto
  • Body: system-ui, -apple-system, Segoe UI, Roboto
  • Mono (code labels): ui-monospace, SFMono-Regular, Consolas

Fonts are intentionally simple to keep performance strong and avoid external font requests.

6) Wireframes

Mobile Home (portrait)

[HEADER: logo + nav (hamburger)]
[HERO image + headline]
[CTA button]
[SECTION: Featured guides (stacked cards)]
[SECTION: Next event]
[FOOTER: links + references]

Desktop Home (wide)

[HEADER: logo | nav links]
[HERO: image left | headline + CTA right]

[ROW: next event card | saved favorites card]
